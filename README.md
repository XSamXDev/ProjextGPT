# ProjectGPT

## Project Documentation

### Features
- Comprehensive AI-driven project proposal generation
- User-friendly interface for interaction
- Supports various project types

### Installation
1. Clone the repository: `git clone https://github.com/XSamXDev/ProjextGPT.git`
2. Navigate to the project directory: `cd ProjextGPT`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

### Usage
- Open the application in your web browser to access the AI project generator.
- Follow the prompts to input your project requirements.

### Tech Stack
- **Front-end**: React.js
- **Back-end**: Node.js, Express
- **Database**: MongoDB
- **AI**: OpenAI API
